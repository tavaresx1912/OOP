# 🎵 Java Music Streaming System

A terminal-based music management system developed in Java as a semester project. The application allows users to register, list, and search songs through an interactive menu.

## Features

- **Add Song** — Register a song with title, artist, duration (in seconds), and genre
- **List Songs** — Display the full catalog of registered songs
- **Search by Title** — Find songs by partial or full title match
- **Duration Formatting** — Automatically formats duration from seconds to `MM:SS`

## Technologies

- Java
- ArrayList (Collections Framework)
- Scanner (user input)
- Object-Oriented Programming

## How to Run

1. Make sure you have the [JDK](https://www.oracle.com/java/technologies/downloads/) installed (Java 8 or higher).
2. Clone the repository:
   ```bash
   git clone https://github.com/your-username/your-repo.git
   ```
3. Compile the file:
   ```bash
   javac StreamingMusic.java
   ```
4. Run the program:
   ```bash
   java StreamingMusic
   ```
